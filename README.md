## 📖 Overview

An interactive HTML5 visual canvas showcasing the **Aries (♈) zodiac** as a futuristic volcanic city — built with pure HTML, CSS animations, Tailwind CSS, and vanilla JavaScript.

**Key Features:**
- 🌋 Animated lava river with flowing gradient
- 🔥 Dynamic ember particle system (JS-generated)
- 🏙️ CSS-built city skyline with neon effects
- ⚡ Interactive energy intensity controls (Calm / Active / Overdrive)
- 📱 Fully responsive (mobile-first)
- ♈ Aries constellation SVG with animated stars

## 🚀 Deployment

This project is deployed automatically via **GitHub Actions** to GitHub Pages on every push to `main`.

### Manual Local Preview

Simply open `index.html` in a browser via a local server (required for `fetch()` component loading):

```bash
# Using Python
python -m http.server 8080

# Using Node.js npx
npx serve .
```

Then visit `http://localhost:8080`
